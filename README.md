## Kuve
Easily view important debugging information for a namespace over multiple environments.

## Installation
- $ brew update
- $ brew install crystal-lang
- $ cd kuve
- $ make update

### To view namespace info
- $ kuve <namespace_of_choice>

### To view node health
- $ kuve nodes
